NetworkModule

object NetworkModule

Module which provides all required dependencies for the network.

Object: Singleton Instance see The Kotlin reference Retrofit: Library used for REST connections. See The Retrofit reference What is Dependency Injection? See this video

Methods annotated with @Provides informs Dagger that this method can provide a certain dependency. Methods annotated with @Singleton indicate that Dagger should only instantiate the dependency once and provide that some object on further requests.
